== Description ==

Pimex is a Web application that lets you organize your potential customers come from wherever it comes, Facebook, Google, AdWords

* Organizes all contacts you accomplished in your web site
* Organizes all the leads attained with your guideline in AdWords
* organizes all of the leads attained with your guideline on social networks
* Send an automatic email after each record.
* You receive a notification when you have a potential customer.
* Identifies the channel of arrival of your potential customers.
* Get graphs of monitoring
* Check your contacts when you contact them.
* Export your contacts with a single click.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload plugin's folder to directory /wp-content/plugins/, or install the plugin through the WordPress plugins screen directly.
2. To use Pimex, you must have installed the plugin Ninja Forms.
3. Activate Ninja Forms through the ‘Plugins’ screen in WordPress.
4. Activate Pimex through the ‘Plugins’ screen in WordPress.
5. Go to “Ninja Forms” menu on WordPress and change the names of the fields in the form: name, email, phone (if you have), and message. In other cases such as custom fields you have to call it “custom-myfield”, for example “custom-city”.
6. Add action Pimex in your Ninja Forms and add your project ID and Token for this form.
7. Just Save and enjoy Pimex.


== Frequently Asked Questions ==

= Could I active this plugin without Ninja Forms? =

No, Ninja Form generate a form for Pimex. If you have another form for contact, you must replace with “Ninja Forms”.

== Screenshots ==

1. Go to “Ninja Forms” menu and select your form.

2. Go to “Emails and Actions” menu and add new action, click in the icon-button “+”.

3. Look a new Action called “Pimex”.  Select it.

4. Add your ID and Token.
